Usando links externos

Você pode acessar o repositório público do professor Gustavo Guanabara no Github
(Aqui estou usando o target = "_blank" e rel = "external" na tag <a>. O que, nesse caso, vai fazer com que eu acesse o link através de uma nova aba, mantendo meu site aberto na aba atual)

Você também pode acessar o canal gratuito do professor no YouTube
(Aqui não estou usando o target = "_blank" etc. O que, nesse caso, vai fazer com que eu acesse o link a partir dessa página, saindo do meu site para entrar no outro)

O ideal é que se você está indicando um link que direciona o usuário para outro site, que não é o seu, esse link seja aberto em outra aba. Ou seja, utilizando o parâmetro target com o valor "_blank" e o parâmetro rel com o valor "external".
