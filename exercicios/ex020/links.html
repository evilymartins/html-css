Personalizando Links

Ao passar o mouse você ativa o :hover
Ao clicar você ativa o :active
Ao visitar você ativa o :visited

Veja o código CSS configurado para isso:


        body {
            font-family: Arial, Helvetica, sans-serif;
        }
        
        a {
            color: blue;
            text-decoration: none;
            font-weight: bold;
        }

        a:visited {
            color: red;
        }

        a:hover {
            background-color: yellow;
            color: black;
            text-decoration: underline;
        }

        a:active {
            background-color: green;
        }

Explicando:

  1. Ao passar o mouse no link você ativa o :hover que vai fazer ele ficar com background-color amarelo, color preto e sublinhado.
  2. Ao clicar você ativa o :active que vai fazer ele ficar com o background-color verde.
  3. Ao visitar você ativa o :visited que vai fazer o color ficar vermelho.
